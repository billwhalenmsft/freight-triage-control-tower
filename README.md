# Freight Triage Control Tower — Prototype

A leave-behind package for **TE Connectivity**, prepared by the Microsoft Solution Engineering team.

**Start with [`index.html`](./index.html)** — it links to everything:

| Item | What it is |
|---|---|
| `index.html` | Landing page — walkthrough video, try-the-prototype, and download links |
| `prototype/` | A pre-built, runnable version of the app (open `prototype/index.html` in a browser) |
| `te-freight-triage-control-tower-project.zip` | The source code to run it yourself (`npm install` → `npm run dev`) |
| `project/` | The same source, unzipped |
| `video/` | Recorded walkthrough (added shortly) |

## The prototype in one line

One surface for the operations team to **Queue → Assign → Resolve** post-booking freight emails —
review the email thread on each item, and track ageing + SLA — running on sample data so you can
explore it immediately.

> Demo data only. No production or customer data is included. Shared in confidence for evaluation.
